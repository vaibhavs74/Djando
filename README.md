# 🚀 Django Web Application

This is a Django-based web application designed to provide multiple cybersecurity and utility features such as password strength checking, phishing URL detection, and scam detection.

## 🔥 Features

- 🔐 User Authentication (Signup/Login)
- 🧠 Password Strength Checker
- 🌐 Phishing URL Detection
- ⚠️ Scam Detection System
- 🎨 Clean and responsive UI

## 🛠️ Tech Stack

- Python
- Django
- HTML, CSS
- SQLite (default database)
